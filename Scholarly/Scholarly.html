Assignment

HCI Homework

Consider as a HCI project the software application to be developed in order to achieve your BSc (Bachelor of Science) title.
The project requires to envision and design an interactive solution based on the existing HCI principles, methods, and best practices

Main Menu

The main menu is the first screen the user sees. He is welcomed by some buttons on the left of the screen, his current car on the right
side of the screen, and a hotbar at the bottom of the scree.

The buttons on the left are: Online game, Options and Exit.

Online game: If the user wants to play the game he presses this buttons and is faced with another menu, the "Play Menu".
Options: If the user wants to change the options of the game, such as sound, video or gameplay,
he can press this button and is faced with the Options menu.
Exit: If the user wants to exit the game, he presses this button.

Options Menu

The options menu is where the user can change the game's settings, such as sound, video or gameplay.
The user sees a rectangle in the middle of the screen, that is scrollable.
Here he sees the options in the order: sound, video, gameplay, sepparated by a line to highlight the category of the options.
The user can go back to the main menu by pressing the x button in the top-right corner of the menu.

Online Game Menu

The online game menu is where the user can select the gamemode that the user wants to play.
The user sees a rectangle in the middle of the screen, with 3 big buttons that show an image
of the gamemode and below it a text with the name of that gamemode.
The gamemodes are: Capture the Flag, Free-for-all, King of the Hill.
The user can go back to the main menu by pressing the x button in the top-right corner of the menu.
